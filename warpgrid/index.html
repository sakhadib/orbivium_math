What and How !?

The Time Warp Visualization is an interactive grid where each grid point represents a "clock" that rotates based on a mathematical function. This creates a dynamic, mesmerizing visual effect that reflects mathematical concepts like oscillations and transformations.

Mathematical Basis

The core of the Time Warp Visualization relies on mathematical functions that take coordinates of grid points (x, y) as inputs and produce an angle of rotation for each clock.

Equations

\( \text{Angle} = f(x, y) \times \text{Speed Multiplier} \)
  • Sine Wave: \( f(x, y) = \sin(x + y) \)
  • Logarithm: \( f(x, y) = \log(x^2 + y^2 + 1) \)
  • Exponential: \( f(x, y) = e^{(x - y)} \)
  • Custom Function: Any user-defined function using x and y.

Example: Sine Wave

Suppose the grid point is at \((x, y) = (1, 2)\) and the speed multiplier is 5.
The angle is calculated as:
\[ \text{Angle} = \sin(1 + 2) \times 5 = \sin(3) \times 5 \approx 0.1411 \times 5 = 0.7055 \text{ radians} \]

Grid and Animation

The grid consists of a defined number of rows and columns. Each point on the grid is represented as a clock. For smaller grid sizes, clocks are drawn with circular shapes. For larger grid sizes (e.g., >30), only the hand directions are displayed for performance optimization.

Animation

Over time, each clock continuously rotates based on the calculated angle and the passage of time. This creates a smooth animation where the entire grid appears to "warp" dynamically.

Equation for Continuous Rotation

\( \text{Animated Angle} = \text{Base Angle} + \text{Time} \)

Example

Grid Size: \(10 \times 10\)

Function: \( \sin(x + y) \)

Speed Multiplier: \(3\)



For a grid point \((x, y) = (2, -1)\):

  • \( \text{Base Angle} = \sin(2 + (-1)) \times 3 = \sin(1) \times 3 \approx 0.841 \times 3 = 2.523 \text{ radians} \)
  • \( \text{Animated Angle} = \text{Base Angle} + \text{Time} \)

Customization Options

Users can interact with the visualization by changing:

  • Grid Size: Number of rows and columns.
  • Function: Predefined or custom mathematical functions.
  • Speed Multiplier: Controls the rotation speed.

Practical Applications

This visualization has applications in:

  • Mathematics Education: Demonstrating waveforms, logarithmic growth, and other mathematical phenomena.
  • Data Visualization: Showing spatial transformations and oscillations.
  • Art and Design: Creating interactive and generative art.
